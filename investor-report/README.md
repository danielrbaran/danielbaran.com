# StudioBaran Investor Report

A professional investor reporting website for StudioBaran, deployed via Vercel.

## Setup

1. Initialize git repository:
```bash
git init
git add .
git commit -m "Initial commit"
```

2. Create a GitHub repository and push:
```bash
git remote add origin [your-github-repo-url]
git branch -M main
git push -u origin main
```

3. Deploy to Vercel:
   - Go to [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Vercel will auto-detect it's a static site
   - Deploy!

## Updating the Report

Simply edit `index.html` and push to GitHub. Vercel will automatically redeploy.

## Structure

- `index.html` - Main report page
- `styles.css` - Styling
- `README.md` - This file

## Customization

Update the content in `index.html` to reflect current status, milestones, and progress.
